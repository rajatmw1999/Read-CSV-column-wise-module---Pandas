STEPS TO SETUP AND USE readcsv.py


1. Create a virtual environment in python and activate it

--->   py -m venv env
--->   .\env\Scripts\activate

2. Install dependencies

--->   pip install -r requirements.txt

3. Using the file

--->   py readcsv.py

4. To deactivate the virtual environment 

--->   deactivate



NOTE: CREATING THE VIRTUAL ENVIRONMENT AND ACTIVATING MAY VARY FROM OS TO OS(LINUX/WINDOWS/MAC), ABOVE MENTIONED IS FOR WINDOWS,
